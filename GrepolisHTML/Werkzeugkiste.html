Nützliche Werkzeuge

Berechnungen zu Laufzeiten

Zur Berechnung von Abfahrtszeiten müssen häufig Zeiten addiert oder subtrahiert werden. Die lästige Rechnerei kann den folgenden Zeitrechner erledigt werden.
Zeitrechner
Weniger offensichtlich nützlich ist das Umrechnen von Laufzeiten unterschiedlicher Geschwindigkeit. Wem jedoch ein Nutzen einfällt, wie beispielsweise die Vervollständigung fehlender Angaben der Mitspieler, kann das folgende Werkzeug benutzen.
Laufzeitrechner
Die Laufzeitumrechnung kann ebenfalls zu einem komfortablen Truppenbewegungsplaner verwendet werden. Durch die Verwendung von ihm genügt es für eine gegebene Strecke die Laufzeit eines Einheitentypes anzugeben. Die restlichen Laufzeiten und Abfahrtszeiten können dann automatisch berechnet werden.
Truppenbewegungsplaner

Berechnungen zur Moral

In den meißten Welten von Grepolis spielt die Moral eine Rolle. Die Moral bei einem Kampf zweier beliebiger Spieler kann mit dem folgenden kleinen Werkzeug berechnet werden.
Moralrechner

Kampf

Mit dem folgenden kleinen Programm kann man die Verteidigungswerte einer fiktiven Abwehr zusammenstellen. Man sieht, dass für eine ausgeglichene Abwehr mehr nötig ist, als nur alle Verteidigungswerte anzugleichen.
Deffensivanalyse
Mit dem folgenden kleinen Werkzeug kann die Abwehr in einer Stadt analysiert werden. Das Ergebnis ist nützlich, um zu entscheiden mit was für einer Armee angegriffen werden sollte.
Einheitenrechner
Im Gegensatz zum vorherigen Werkzeug, kann mit diesem Werkzeug die Zusammensetzung eines Angriffs ermittelt werden.
Angriffsanalyse
Abhilfe ist mit diesem kleinen Programm gegeben. Mit dem Programm kann die Zusammensetzung einer ausgeglichenen Abwehr ermittelt werden.
Rechner zur Deffensivzusammensetzung
Eine Erweiterung dieses Programm ist hier zu finden. Über die Felder Offset können weitere Verteidigungswerte wie Miliz und mythische Einheiten in die Zusammensetzung miteinbezogen werden.
erweiterter Rechner zur Deffensivzusammensetzung
In den vorherigen Programmen wurde stets davon ausgegangen, dass die Armee des Angreifers in ihrer Stärke der Armee des Verteidigers entspricht. Inwieweit sich die Verlustrate bei anderen Werten ändert, kann mit dem folgenden Programm getestet werden.
Verlustrechner

Stadtplanung

Mit dem folgenden kleinen Programm kann bei einer gegebenen Anzahl von Bauernhofplätzen die Zusammensetzung einer ausgeglichenenen Abwehr berechnet werden und wieviele Transportschiffe zum Überseetransport benötigt werden.
Deffensivstadtrechner

Organisation
